# MindForge AI Exam 2 - Student Performance Analytics

---

# Project Objective

The objective of this project is to analyze student performance data using Python and data analysis techniques. The project calculates a readiness score for each student based on attendance, diary completion, task performance, assignment marks, and lab performance. It also classifies students into different performance levels and generates visual insights to support academic decision-making.

---

# Technologies Used

* Python 3
* Jupyter Notebook
* Pandas
* NumPy
* Matplotlib

# Features

* Load or create the student dataset
* Display dataset preview
* Check data types and missing values
* Calculate student readiness score
* Classify students into:

  * Support
  * Developing
  * Strong
  * Excellent
* Generate charts for analysis
* Provide final insights and interpretation

---

# How to Run

1. Clone this repository.
2. Open the `notebooks` folder.
3. Launch `student_performance_analysis.ipynb` using Jupyter Notebook or Google Colab.
4. Run all cells from top to bottom.
5. Review the generated tables, charts, and final insights.
