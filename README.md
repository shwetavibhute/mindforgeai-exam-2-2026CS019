# Objective

The objective of this project is to analyze student performance data using Python and Pandas. The analytics pipeline validates the dataset, calculates a readiness score based on academic performance indicators, classifies students into performance categories, generates visualizations, and provides meaningful insights to support academic decision-making.

# How to Run the Notebook

1. Clone or download this repository.
2. Open the project in Visual Studio Code or Jupyter Notebook.
3. Navigate to the notebook in the `Section_C/notebooks` folder.
4. Ensure the required Python libraries are installed:

   * pandas
   * numpy
   * matplotlib
5. Run all notebook cells from top to bottom.
6. Review the generated tables, charts, and final insights.
